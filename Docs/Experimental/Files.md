# json2yy

Transforms a JSON input into a YY string used by GameMaker. Thise function is not recommended to be used directly if not modifying GameMaker's .YY files.

### Syntax

```js
json2yy(json);
```

| Argument |  Type  | Description |
| :------- | :----: | :---------- |
| json     | String | The JSON    |

### Returns

```js
String;
```
